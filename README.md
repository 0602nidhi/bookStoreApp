## Getting Started
These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Running the website
1. Clone the repository.
2. Open it on VSCode.
3. Open new terminal and split it into two parts.
- part 1 - Frontend

```
cd frontend
npm install
npm run dev
```

- part 2 - Backend

```
cd backend
npm install
npm start
```
